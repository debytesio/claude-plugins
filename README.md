# DeBytes Claude Code Plugins

AI-powered plugins for [Claude Code](https://claude.ai/claude-code) by [DeBytes](https://debytes.io).

## Installation

Add the DeBytes marketplace to Claude Code:

```
/plugin marketplace add debytesio/claude-plugins
```

Then install any plugin:

```
/plugin install job-hunter@debytes
```

## Available Plugins

| Plugin | Description | Status |
|--------|-------------|--------|
| [job-hunter](https://github.com/debytesio/claude-plugin-jobhunter) | AI-powered job search across UK and France platforms with visa sponsorship checking, salary analysis, and ranked Excel exports | Available |
